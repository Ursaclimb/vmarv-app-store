# vMarv's Umbrel App Store

A community app store with working applications for Umbrel OS.

## How to Add This App Store

1. Navigate to the **App Store** section in your Umbrel dashboard
2. Click the "..." menu in the top right corner and select **Community App Stores**
3. Paste the following URL and click **Add**:

```
https://github.com/vmarv/vmarv-app-store
```

The apps will then be available in your Umbrel App Store.

### Litecoin Node
A fully functional Litecoin node with:

- Pre-configured with mining credentials
